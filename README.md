

# Simple Portfolio Website

A clean, responsive personal portfolio website built using HTML5, CSS3, and Vanilla JavaScript. This project serves as a professional landing page to showcase technical skills, academic projects, and contact information.

## Live Demo
View the live project here: [https://goutham-m.vercel.app/](https://goutham-m.vercel.app/)

## Features
* Responsive Design: Optimized for various screen sizes, including desktop and mobile.
* Skills Section: Detailed display of technical competencies and tools.
* Projects Gallery: A structured layout showcasing recent web development and software projects.
* Functional Contact Form: Integrated "Send Now" button for direct communication.
* Downloadable Resume: Quick access to a professional CV in PDF format.

## Technologies Used
* HTML5: Structural layout and content.
* CSS3: Custom styling, animations, and responsive grids.
* JavaScript: Interactive UI elements and form handling.

## Project Structure
* /images: Directory containing all graphical assets and skill icons.
* index.html: The primary entry point for the website.
* style.css: The main stylesheet for layout and design.
* script.js: JavaScript logic for interactivity.
* resume.pdf: Professional documentation for career inquiries.

## Installation and Local Setup
1. Clone the repository:
   git clone [https://github.com/gouthamkulall615-art/simple-portfolio-using-HTML-and-CSS.git](https://github.com/gouthamkulall615-art/simple-portfolio-using-HTML-and-CSS.git)

2. Navigate to the project directory:
   cd simple-portfolio-using-HTML-and-CSS

3. Run the project:
   Open index.html in any modern web browser to view the site locally.
